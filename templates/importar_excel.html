Instrucciones
Formato del Archivo Excel:

Tu archivo Equipos.xlsx debe tener las siguientes columnas:

  • EQUIPOS - Nombre del equipo (obligatorio)
  • FECHA - Fecha del mantenimiento
  • REPUESTOS - Repuestos utilizados
  • MANO DE OBRA - Trabajo realizado
Notas Importantes:
  • El archivo debe estar en el mismo directorio que la aplicación
  • Se importarán solo las filas que tengan nombre
  • Los equipos importados se marcarán como "Activos"
  • Si hay equipos duplicados, se crearán registros adicionales
Archivo Detectado: Equipos.xlsx

Haz clic en el botón para importar los datos del archivo Excel a la base de datos.

¿Qué pasará cuando importes?
Se leerán todos los datos del archivo Excel y se agregarán como nuevos equipos en la base de datos. Podrás ver y editar los equipos importados desde la sección "Equipos".

Esta operación es segura y no sobrescribirá datos existentes.
